# Nowplay Front-end
This is the front-end of Nowplay project. For working with this app you first need to start up the [back-end](https://github.com/kiano0sh/nowplay-backend).

## Setup

### Requirements
- Node.js
- Npm
- react-native
- AVD or Real Device

## Install

```sh
$ npm i
```
### How does it work
```sh
$ react-native run-android
```
